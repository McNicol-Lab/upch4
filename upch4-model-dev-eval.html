UpCH4: Random Forest Model Dev. and Eval.

Gavin McNicol

2023-05-24

Purpose

Code for predictive model development and evaluation for upscaling wetland eddy covariance methane flux data from sites to the globe. The code takes pre-processed weekly mean eddy covariance flux tower from FLUXNET-CH4 v1.0 (Delwiche et al.2021) and static and dynamic geospatial predictor datasets as input for a random forest model training pipeline, and outputs:

  1. a best-predictor set using forward spatial feature selection
  2. spatial cross validation metrics of model performance
  3. variable importance rankings

The final three sections output:

  1. simulated training datasets from Monte Carlo to propagate uncertainties
  2. representativeness products (dissimilarity and constituency maps)
  3. globally upscaled product evaluations

Manuscript Reference: McNicol, G. Fluet-Chouinard, E. et al.Upscaling wetland methane emissions from the FLUXNET-CH4 eddy covariance network (UpCH4 v1.0): Model development, network assessment, and budget comparison. AGU Advances, Accepted, May 2023

FLUXNET-CH4 Data

The eddy covariance data used in this study are publicly available as part of the FLUXNET-CH4 community product V1.0 at fluxnet.org. The FLUXNET-CH4 synthesis activity is introduced in Knox et al.2019 along with a detailed description of the eddy covariance post-processing steps including methane flux (FCH4) uncertainties. The first full (V1.0) dataset release (FLUXNET-CH4, hereafter) is described for 81 sites and is used in a wetland seasonality analysis in Delwiche et al.in review. Data for this CH4 Upscaling Project were downloaded from fluxnet.org on Feb 22, 2021. Download Manifest
